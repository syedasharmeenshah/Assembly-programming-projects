🧮 Vowel Quantitative Tabulation Program (Assembly Language)

This project is a low-level Assembly Language program developed as part of a university coursework assignment. The program takes a user-input string and performs character-level analysis to count the total number of vowels present in the input.

It demonstrates fundamental concepts of system-level programming, including direct memory manipulation, interrupt-based I/O handling, and register-level operations.

🚀 Features
Accepts a string input from the user
Scans each character using low-level memory access
Identifies both uppercase and lowercase vowels (a, e, i, o, u)
Calculates total vowel count using CPU registers
Displays the final result using Linux system calls
⚙️ Technical Highlights
Written in x86 Assembly Language
Uses Linux int 0x80 system calls for input/output
Direct register manipulation (eax, ebx, ecx, esi, bl)
Efficient loop-based character traversal
ASCII conversion for output display
🧠 Learning Outcomes

Through this project, I strengthened my understanding of:

Low-level programming concepts
Memory addressing and data handling in assembly
System calls and OS-level interaction
Control flow using jumps and conditional logic
Character processing at byte level
